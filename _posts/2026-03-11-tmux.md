## tmux command

```bash
# 关闭指定对话
tmux kill-session -t x

# 一次性关闭所有session
tmux ls | awk -F: '{print $1}' | xargs -I {} tmux kill-session -t {}

```

## tmux internal command

| command      | describe          |
| ------------ | ----------------- |
| bind-key + , | 重命名window         |
| bind-key + [ | 复制模式; 使用vim键位移动光标 |

## 复制模式

例如我的控制键为：C-b

1. `C-b [` 进入复制模式
2. 参考上表移动鼠标到要复制的区域，移动鼠标时可用vim的搜索功能”/“,”?”
3. 空格键开始选择复制区域
4. 选择完成后安enter键退出
5. `C-b ]` 粘贴

## 强制重绘当前 Pane

这是最直接和常用的方法，它会告诉 tmux 重新计算并绘制当前活动的 pane。

1. **进入 tmux 命令模式：** 按下 `Ctrl+b` 然后松开，再按下 `:` (冒号)。 此时，tmux 底部会出现一个命令行提示符。
2. **执行 `refresh-client` 命令：** 在命令行中输入 `refresh-client -S` 并按下回车。
    - `refresh-client`：这个命令用于刷新客户端的显示。
    - `-S`：这个选项是关键，它告诉 tmux 重新计算并设置当前客户端的屏幕大小。
