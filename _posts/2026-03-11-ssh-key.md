# 创建

```bash
ssh-keygen
```

# 传输

1. 使用ssh-copy-id传递sshkey
    ```bash
    ssh-copy-id -i ~/.ssh/id_rsa.pub  user_name@x.xxx.x.xxx
    ```
2. 在windows中可以这样做
    ```powershell
    type $env:USERPROFILE\.ssh\id_rsa.pub | ssh {IP-ADDRESS-OR-FQDN} "cat >> .ssh/authorized_keys"
    ```
3. 使用scp传递秘钥
    ```bash
    scp ~/.ssh/id_rsa.pub user_name@x.xxx.x.xxx:~/tmp_sshkey

    cat ~/tmp_sshkey >> ~/.ssh/authorized_keys
    ```
